# Set Up 
## Frontend Set Up
```
cd gridaan-assessment
``` 
```
pnpm install
``` 
```
npm run corepack:enable
``` 
```
cd frontend
``` 
```
npm run dev
``` 
## Backend Set Up
```
cd gridaan-assessment
``` 
```
pnpm install
``` 
```
npm run corepack:enable
``` 
```
cd backend
``` 
```
copy .env.example .env
```
Add your keys in .env file
```
npm run dev
```
